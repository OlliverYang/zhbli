控制列间距：

```latex
\setlength{\tabcolsep}{4pt}
```

有序列表：

```latex
\begin{enumerate}
\item This is the first item
\item This is the second item
\item This is the third item
\end{enumerate}
```

无序列表：

```latex
\begin{itemize}
\item This is the first item
\item This is the second item
\item This is the third item
\end{itemize}
```

Multi-column cells：

```latex
\multicolumn{number cols}{align}{text} % align: l,c,r
```
